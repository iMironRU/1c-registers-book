# Регистры 1С: от скелета до осознанного выбора

> Серия **«1С как иностранный язык»**. Книга о регистрах платформы «1С:Предприятие 8» —
> от общего скелета через бытовые метафоры до осознанного выбора, **без перегруза бухгалтерией
> до самой последней фазы**.

**Автор:** [iMironRU](https://github.com/iMironRU) · **Лицензия:** [CC-BY-SA 4.0](LICENSE) · **Статус:** в работе (v0.1.0)

---

## Для кого

Первокурсник, прошедший предыдущие книги серии. Умеет читать BSL и метаданные, знает супра-метафору
*существительное→справочник, глагол→документ, **регистр→книга учёта***. Регистры в учебнике видел,
в практике — почти нет.

**Не для:** бухгалтеров, расчётчиков, опытных 1С-разработчиков, читателей вне серии.

Подробнее — [`spec/specification.md`](spec/specification.md).

---

## Принципы (контракт с читателем)

1. **Граница доменов жёстко.** Платформа (как хранится/считается) и предметка (что значит) —
   не смешиваются. Уровни 0–1 — только платформа.
2. **Упрощение = частный случай, не противоречие.** Позднее уточнение достраивает раннее, не сносит.
3. **Термин — награда, а не входной билет.** Порядок: *пример из жизни → мостик → ярлык*.
4. **Учим на родном домене ученика.** Регистр не знает, что он «бухгалтерия».
5. **Цена механизма.** Сложнее регистр → больше возможностей и дороже сопровождать.

Полный контракт — [`spec/constitution.md`](spec/constitution.md).

## Сборка

```bash
./book.sh status           # прогресс по статусам параграфов
./book.sh build            # все форматы из metadata.yaml
./book.sh build ready      # только финальные главы
./book.sh release          # релиз с обновлением версии и changelog
```

Требования: `pandoc`, `python3` + `pyyaml`, `mdbook`, `xelatex` (для PDF).
